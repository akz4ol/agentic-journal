# Submission Template

## How to Submit

1. **Fork the Repository**
   - Fork [agentic-journal](https://github.com/akz4ol/agentic-journal)

2. **Create Your Submission**
   - Copy this `SUBMISSION_TEMPLATE` folder
   - Rename it to your submission (e.g., `smith-2025-neural-agents`)
   - Fill in `metadata.yaml` with your paper details
   - Add your manuscript as `paper.pdf`

3. **Submit via Pull Request**
   - Create a PR from your fork to the main repository
   - Target the `main` branch
   - Title: "Submission: [Your Paper Title]"

4. **Automated Review**
   - Our multi-agent system will automatically review your paper
   - You'll receive feedback as PR comments within 24-48 hours
   - Possible outcomes:
     - **Accept**: Paper accepted for publication
     - **Minor Revision**: Small changes needed, resubmit
     - **Major Revision**: Significant changes needed
     - **Reject**: Does not meet publication criteria

## Requirements

### Manuscript (paper.pdf)
- PDF format only
- 4-30 pages (depending on paper type)
- Maximum 50MB file size
- Include figures/tables in the PDF

### Metadata (metadata.yaml)
- All required fields completed
- Abstract: 150-300 words
- Valid author emails
- Valid paper type

## Paper Types

| Type | Description | Page Limit |
|------|-------------|------------|
| research | Original research with experiments | 8-25 |
| position | Opinion/perspective pieces | 4-12 |
| survey | Literature reviews | 15-30 |
| technical | Technical reports, tools | 6-20 |
| tutorial | Educational content | 8-20 |
